Beautifulhugo Theme Extras - Chroma Server Side Syntax Highlighting

Published: Tue 11 August 2020
By Michael B

In Hugo.

tags: hugo beautifulhugo chroma CSS "static files"

The beautifulhugo theme has support for Hugo’s Chroma (a general purpose syntax highlighter in pure Go) syntax highlighting.

Let me try that out! 🚀

Add the following to the site config i.e.config.toml file:

pygmentsCodeFences = true
pygmentsUseClasses = true

Then from the command line, generate the desired new code style (in my case, monokai) by running:

hugo gen chromastyles --style=monokai > static/syntax.css

Or if you want to add more structure to your static files, add the following to the site config i.e. config.toml file:

staticDir = ["static", "static/css"]

And then re-generate the code style using the new folder path i.e. static/css :

hugo gen chromastyles --style=monokai > static/css/syntax.css
