Beautifulhugo Theme Extras - Shortcodes (Details)

August 12, 2020 ¬∑ 2 min ¬∑ Michael B

An extra beautifulhugo theme shortcode is called details (checking the code itself shows that it is based on the HTML <details> tag). This shortcode hides some content until the text set on the detail is clicked.

An example follows:

Click here to expand to show hidden content!

This is the content you’d like to hide. üßò

The actual code used in this post for the previous example looks like this:

1
2
3

{{< details "Click here to expand to show the content!" >}}
This is the content you'd like to hide. üßò
{{< /details >}}

I had trouble adding spacing to the content in the details shortcode above so I resorted to adding HTML <br> tags but that didn’t work either. Probably need to escape some HTML or something like that - not looking into that now though - maybe for another time.

Curiously also, I found that it is impossible (for me) to add extra whitespace after the details shortcode. I’ve tried adding 1 or more blank lines inside of <pre></pre> tags and I tried adding numerous <br> tags after the details shortcode but neither of these approaches worked. In fact when I add a <br> tag after the shortcode the line after is not displayed … not sure what is going on.

The learning continues! üèÉ
