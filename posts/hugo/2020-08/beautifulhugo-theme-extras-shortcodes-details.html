Beautifulhugo Theme Extras - Shortcodes (Details)

Published: Wed 12 August 2020
By Michael B

In Hugo.

tags: hugo beautifulhugo shortcodes

An extra beautifulhugo theme shortcode is called details. This shortcode hides content until the text set on the detail is clicked.

The example code looks like this:

{{</* details "Click here to expand to show the content!" */>}}
<br>
&nbsp;&nbsp;&nbsp;This is the content you'd like to hide.
<br>
&nbsp;&nbsp;&nbsp;🧘‍♂️
{{</* /details */>}}

I was having trouble adding spacing to the hidden content, above and below “This is the content you’d like to hide.” in the example above, so I resorted to adding HTML <br> tags. The markdown linter I have installed doesn’t like it though - no inline HTML rule broken. Still works.

And I had to add numerous non-breaking spaces to push the content to the right i.e. &nbsp;&nbsp;&nbsp; - there must be a different way!?

Curiously also, I found that it is impossible to add extra whitespace after the details-shortcode. I’ve tried adding 1 more blank lines inside of <pre></pre> tags, and I’ve added numerous <br> tags after the details-shortcode but neither of these approaches worked.

The learning continues! 🏃‍♂
