Escaping Hugo Shortcodes in Posts

Published: Fri 14 August 2020
By Michael B

In Hugo.

tags: hugo beautifulhugo shortcodes

TL;DR

Add /* after the opening shortcode tags i.e. {{</* and before the closing shortcode tags i.e. */>}}

While creating the post Beautifulhugo Theme Extras - Shortcodes (Split), I had quite a bit of trouble finding the answer to the question of “how can I display a block of code in my post?“.

The answer seemed obvious, use a code fence i.e. use triple backticks; OR indent the code block using 4 spaces; OR after adding a colon (:), surround the code by two blank lines, one at the top and one below the last line of code.

As it turned out, that was the wrong question! I, Robot anyone?

The right question was “what is the markdown escape sequence required to correctly display a hugo shortcode in my post i.e. without it being interpreted?“.

The answer eventually came to me via Desmond Lua at LuaSoftware.

Thanks Desmond! 😅
